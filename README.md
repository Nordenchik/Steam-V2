# Steam V2
Steam V2 - це моя видозмінена (і трошки зменшена) версія _Steam_.

![тут має бути картинка steam](steam-logo-steam-icon-transparent-free-png.png)

#### У цьому додатку є такі сторінки:
* Магазин ігор;
* Бібліотека придбаних ігор;
* Спільнота придбаних ігор;
* Ваш профіль;
* Керування завантаженнями ігор та їх оновлень або доповнень.
